## вңЁ Features

### рҹҢӨпёҸ Live Weather
- Fetches real-time weather using Weather API.
- Displays:
  - Temperature
  - Weather Condition
  - Feels Like
  - Humidity
  - Wind Speed
- Automatically updates weather icon.
- Shows current date and live time.
- Uses Geolocation API to detect the user's location.

---

### вң… To-Do List
- Add new tasks.
- Mark tasks as important.
- Mark tasks as completed.
- Delete tasks.
- Saves tasks using Local Storage.

---

### рҹ“… Daily Planner
- Organize your day with hourly time slots.
- Add and edit plans.
- Automatically saves plans using Local Storage.
- Scrollable planner layout.

---

### рҹҺҜ Daily Goals
- Add daily goals.
- Mark goals as completed.
- Delete goals.
- Progress is saved using Local Storage.

---

### вҸұпёҸ Pomodoro Timer
- 25-minute focus timer.
- Start timer.
- Pause timer.
- Reset timer.
- Improve productivity using the Pomodoro Technique.

---

### рҹ’¬ Motivation Quotes
- Fetches inspirational quotes from an API.
- Generate a new quote with one click.
- Handles API errors gracefully.

- Theme preference is saved using Local Storage.
